# [AEE-700] 什麼是執行框架？

## Context

「執行框架（Harness）」是智能體工程中最重要也最缺乏明確定義的術語之一。有些人用它來泛指 LLM 周圍的任何腳手架；有些人特指評估基礎設施。實際上，它涵蓋了使智能體能夠運作的完整軟體基礎設施——模型本身以外的一切。

## Design Think

**執行框架**是圍繞 LLM 的完整軟體基礎設施，使其能夠作為自主智能體運作。它涵蓋模型本身以外的所有程式碼、配置和執行邏輯——包括智能體循環、工具執行、記憶系統、上下文管理、狀態持久化、錯誤恢復和防護欄。

執行框架分為兩個主要子學科：

1. **執行型框架** -- 在生產環境部署智能體的運行時基礎設施（如 Claude Code）
2. **評估型框架** -- 測量智能體能力的標準化測試基礎設施（如 Inspect AI、SWE-bench）
