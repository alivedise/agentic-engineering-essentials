# [AEE-100] 什麼是智能體？

## Context

「智能體」一詞在 AI 討論中被隨意使用——有時指帶有系統提示的簡單聊天機器人，有時指能自主運行數小時的完全自主系統。缺乏共同定義，工程師就無法清晰推理自己在建構什麼，也無法評估系統是否符合目的。

## Design Think

**智能體**是一個感知環境、做出決策、並採取行動以實現目標的系統——可能跨越多個步驟，使用文字生成之外的一個或多個工具或能力。

區別智能體與普通 LLM 呼叫的關鍵屬性：

1. **多步驟推理** -- 系統能規劃並執行一系列行動，而不僅僅是一次性回應
2. **工具使用** -- 系統能對世界採取行動（讀取檔案、呼叫 API、執行程式碼、瀏覽網頁）
3. **狀態** -- 系統在步驟之間維護上下文（對話歷史、記憶、中間結果）
4. **目標導向** -- 系統朝著目標行動，而不僅僅是回答問題

系統越自主地使用這些屬性，其「智能體性」就越強。
