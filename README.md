# Agentic Engineering Essentials (AEE)

A collection of vendor-agnostic agentic engineering principles and best practices, built as a bilingual (English / Traditional Chinese) documentation site.

## Overview

AEE provides numbered, structured guidelines for engineers building, running, and governing agentic AI systems -- from foundational mental models to production harness design, multi-agent orchestration, evaluation, and security. Each article includes context, design thinking, optional deep dives, best practices, diagrams, and examples.

## Development

```bash
pnpm install
pnpm docs:dev      # Start dev server
pnpm docs:build    # Build for production
pnpm docs:preview  # Preview production build
```
